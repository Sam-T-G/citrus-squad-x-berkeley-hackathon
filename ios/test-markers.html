Last-50-feet test markers

Each sheet is one printed anchor for the Citrus Squad final-approach detection layer. The QR encodes the machine payload DOOR:room-NNN that the on-device barcode scanner decodes; the large ROOM NNN text below it is the human-readable sign the Claude honest reader confirms.

To test: print at full size (the QR should be ~5 in), on matte paper, no lamination (glare kills the decode). Tape each at chest-to-eye height. In the app, open the Diagnostics tab, start the camera, tap Scan markers on the Anchors card, and point the phone at a sheet from a few feet away. The payload, bearing, and LiDAR distance should appear. For a real run, print 4–6 of each and cluster them at one decision point so a blind-aimed frame catches at least one. High error correction is on, so a corner can be occluded and it still decodes.
